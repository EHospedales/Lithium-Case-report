# Lithium Journal Club – Interactive Presentation

An interactive journal club presentation for psychiatric residents based on the 2025 *Nature* article on lithium:
**[nature.com/articles/s41586-025-09335-x](https://www.nature.com/articles/s41586-025-09335-x)**

## How to Use

Open `index.html` in any modern web browser — no installation or internet connection required.

### Navigation
| Action | Control |
|---|---|
| Next slide | `→` / `↓` / `Page Down` / **Next** button |
| Previous slide | `←` / `↑` / `Page Up` / **Prev** button |
| Toggle speaker notes | `N` key or **📝 Speaker Notes** button |

### Features
- **18 interactive slides** covering background, pharmacology, study critical appraisal, and clinical implications
- **Expandable accordion sections** for study design and discussion questions
- **Live audience poll** — residents vote and see results in real time
- **Branching case vignette** with immediate teaching feedback
- **Knowledge quiz** with instant answer validation
- **Progress bar** and slide counter
- **Speaker notes** for the facilitator on every slide
- Hover **tooltips** on medical abbreviations and jargon

### Presentation Flow (~60 minutes)
1. **Background** – History &amp; pharmacology of lithium (slides 1–7)
2. **The Study** – Critical appraisal framework for the Nature article (slides 8–10)
3. **Case Discussion** – Lithium toxicity vignette with branching decisions (slides 11–12)
4. **Audience Poll** – Gauge residents' clinical comfort level (slides 13–14)
5. **Knowledge Quiz** – Three pharmacology questions with feedback (slide 15)
6. **Clinical Implications &amp; Discussion** – Take-home points &amp; open questions (slides 16–18)

## Requirements
Any modern browser (Chrome, Firefox, Safari, Edge). No dependencies or build step needed.
